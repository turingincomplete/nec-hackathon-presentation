A NEC Hackathon Presentation

Utilizing reward system and live tracking for better public transportation

Made by - team turingincomplete

A progressive web app that makes public transport more convenient and accessible. It provides assistance with live tracking schedules and a reward system which encourages users towards public transport.

User oriented live-tracking

Instead of using centralized and cost-ineffective methods for real-time location of transports. Our system relies on user focused live tracking made possible by active users providing real-time location of their current route.

Accurately predicts transportation delays

To determine scheduling delays the system uses a time-series algorithms which considers factors like congestion, commute-hours and week-days patterns, route obstructions, etc.

Users get incentivized for using public transport

A reward system that provides incentives to the user for providing their real-time geolocation data which helps to track the particular transport.

Intuitive User Experience

A simple, intuitive and straight-forward user experience with localisation support for mutiple regional languages.
